🆘 QUICK TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

  Problem: "Database not found" error
  Solution: Make sure MySQL is running
           Create database: mysql -u root -e "CREATE DATABASE register_db;"

  Problem: Status doesn't change red/green
  Solution: Refresh browser
           Wait 30 seconds for auto-refresh
           Check timezone in .env file

  Problem: Can't login with test account
  Solution: Run migration again: php artisan migrate
           Run seeder: php artisan db:seed

  Problem: Need to reset everything
  Solution: php artisan migrate:fresh --seed
           (WARNING: This deletes all data!)
