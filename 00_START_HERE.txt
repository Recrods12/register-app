🆘 QUICK HELP

═══════════════════════════════════════════════════════════════════════════

Problem: Database error saat migration
Solution: mysql -u root -e "CREATE DATABASE register_db;"
          php artisan migrate --fresh --seed

Problem: Status tidak berubah merah/hijau
Solution: Check timezone di .env (APP_TIMEZONE=UTC)
          Refresh browser atau tunggu 30 detik

Problem: Lupa password test user
Solution: php artisan tinker
          $user = User::find(1);
          $user->password = bcrypt('password');
          $user->save();

Problem: Butuh reset database (HATI-HATI: HAPUS DATA)
Solution: php artisan migrate:fresh --seed
